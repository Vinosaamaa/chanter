# Interactive product tour (Cursor canvas)

`chanter-product-vision.canvas.tsx` is a **clickable screen walkthrough** of the final Education MVP. It covers marketing home, auth, Study Server shell, `#questions`, voice, Friends hub, instructor dashboard, and later phases.

## Open in Cursor

1. Copy or symlink this file into your workspace Cursor canvases folder:
   `~/.cursor/projects/<workspace>/canvases/chanter-product-vision.canvas.tsx`
2. Open the `.canvas.tsx` file in the IDE.
3. Use **Open beside chat** to interact — click action buttons to move between screens and read user stories.

The canonical copy for the repository lives here under `docs/product-design/interactive/`. Update this file when the product vision changes, then refresh the canvases copy if you use the live tour.

## What it is not

This is a **product narrative tool**, not the production frontend. Implementation lives in `frontend/` and follows vertical slices in [issue breakdown](../../issues/education-mvp-issue-breakdown.md).
